Basic Command

Here are the basic commands used in git:

  • git clone <url> [<local directory>]

    clone git project from an url to local repository

  • git add <file or directory name>

    add file or all the directory content to next commit

  • git commit [-m <message>]

    add file or all the directory content to next commit
